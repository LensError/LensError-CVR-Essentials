# LensErrors CVR Essentials

A collection of essential prefabs and assets for ChilloutVR.

## Installation

**Option 1 — Full package (ZIP)**
Download this repository as a ZIP, extract it, and drop the contents into `Assets/LensError-CVR-Essentials/` in your Unity project.

**Option 2 — Pick what you need (UnityPackage)**
Head to the [Releases](#releases) table, grab whichever package you want, then double-click the `.unitypackage` or drag it straight into your project.

## Requirements

Before importing this package into your Unity project, make sure you have the following installed:

- **[Blender 5.0.1+](https://www.blender.org/download/)** - 3D models are distributed as `.blend` files instead of FBX for easier editing. Blender must be installed via the installer, portable Blender will not work. Without a proper Blender installation, Unity cannot import them and no 3D models will appear.
- **Latest CVR CCK (ChilloutVR Creator Kit)** - Required for all CVR-specific components. See the [CCK Setup Guide](https://documentation.chilloutvr.net/cck/setup/) for installation instructions.
